# mini-online-code-editor
A mini online code editor that allows user to write the code
OVERVIEW
the mini online code editor provides an interactive environment where user can get live output

Features
live preview of html,css,javascript
separate editors for html,css,javascript
clean and responsive user interface

Tchnologies i used
HTML5
CSS3 
Javascript

How it works
users wrote the code in separate areas for html,css,javascript 
javascript listens for user input events
the written code is dynamically injected into a frame using 'srcdoc
output updates instantly
